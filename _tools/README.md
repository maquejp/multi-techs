# Tools Directory

This directory contains various tools and templates for setting up different types of projects and development environments.

**Path:** `./tools`

## Directory Structure

- `databases/` - Contains Docker Compose files and scripts for setting up different database environments
- `guis/` - Templates and tools for different types of GUI applications:
- `desktop/` - Desktop application templates (Electron, Tauri, etc.)
- `mobile/` - Mobile application templates (React Native, Flutter, etc.)
- `web/` - Web application templates (React, Vue, Angular, etc.)
