# POSTGRESQL

This directory contains the scripts and the configuration file for instantiating a docker container hosting a postgresql server

[POSTGRESQL](https://www.postgresql.org/)

**Path:** `./tools/databases/postgresql`

## Usage

```bash
./start-postgresql.sh
```
