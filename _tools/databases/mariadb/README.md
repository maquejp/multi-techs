# MariaDB

This directory contains the scripts and the configuration file for instantiating a docker container hosting a mariadb server

[MariaDB](https://mariadb.org/)

**Path:** `./tools/databases/mariadb`

## Usage

```bash
./start-mariadb.sh
```
